Amigo Secreto

Como funciona o sorteio do Amigo Secreto?

O sorteio do Amigo Secreto é uma jogo onde cada participante sorteia o nome de outro participante para presenteá-lo.

Veja como funciona:

  1. Adicione os nomes dos participantes no campo de entrada.
  2. Clique no botão "Sortear amigo" para iniciar o sorteio.
  3. O sorteio seguirá uma ordem definida, então todos devem respeitar sua vez.
  4. Use o botão "Revelar amigo" para ver quem foi sorteado. Certifique-se de que ninguém esteja bisbilhotando.
  5. Você pode reiniciar o sorteio a qualquer momento clicando no botão "Reiniciar".
  6. Caso queira encerrar o sorteio, clique no botão "Parar".
